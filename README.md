# Employee Management System (MERN)

A full-stack personnel management portal built with the MERN stack (MongoDB, Express, React, Node.js) that streamlines employee management, leave requests, attendance tracking, and comprehensive analytics.

## 🚀 Tech Stack

- **Frontend**: React.js, React Router, State Management
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Testing**: 85% Test Coverage

## ✨ Key Features

### Core Functionality
- **Employee Management**: Full CRUD operations for employee records
- **Leave Management**: Submit, approve, and track leave requests
- **Attendance Tracking**: Real-time attendance monitoring and records
- **Dashboard Analytics**: Visual insights on headcount and leave patterns

### Security & Access Control
- **JWT Authentication**: Secure token-based authentication
- **Role-Based Access Control**: Separate interfaces for Admin and Employee roles
- **Admin Dashboard**: Manage employees, approve leave, view analytics
- **Employee Portal**: Request leave, track attendance, view personal data

### Performance & UX
- **React-Based Routing**: Fast, responsive navigation
- **State Management**: Optimized state handling for smooth user experience
- **15% Bounce Rate Reduction**: Improved engagement through better UX
- **85% Test Coverage**: Comprehensive test suite for reliability

## 📋 Prerequisites

- Node.js (v14 or higher)
- MongoDB (local or Atlas)
- npm or yarn package manager
- Git
